// HOW I APPROACH THE WORK

Judgment a résumé can't show.

01 / OBJECTIVE-FIRST

Start from the objective, not the checkbox.

Every control has assessment objectives an assessor actually scores against — abc. I map evidence to each objective, not just the control, so a readiness review survives contact with a real assessor.

02 / DEFENSIBLE N/A

Know when 'Not Applicable' is defensible — and when it isn't.

N/A at Level 1 is rare and must be justified in writing. I document the determination, the boundary logic, and the re-evaluation trigger — e.g., SC.L1-3.13.5 is a legitimate N/A only when there are no publicly accessible components in scope.

03 / TRANSLATE ONCE

Translate, don't duplicate.

Federal and commercial frameworks are the same controls in different dialects. I build the crosswalk once so a team pursuing CMMC and SOC 2 doesn't do the work twice.
